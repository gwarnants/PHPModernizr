PHPModernizr
============

A library of commonly used built-in PHP functions and global constants not available on old PHP versions.

- array_combine()
- array_diff_uassoc()
- array_udiff()
- array_udiff_assoc()
- array_udiff_uassoc()
- array_fill_keys()
- image_type_to_extension()
- imageflip()
- file_get_contents()
- file_put_contents()
- str_split()
- strpbrk()
- substr_compare()
- lcfirst()
- hex2bin()
- scandir()
- boolval()
