PHPModernizr
============

A server-side modernizr library that gives the ability to use last built-in PHP functions on old PHP versions.
Can be helpful to work on old PHP 4 servers.

- apache_response_headers()
- array_column()
- array_combine()
- array_diff_uassoc()
- array_fill_keys()
- array_replace()
- array_replace_recursive()
- array_udiff()
- array_udiff_assoc()
- array_udiff_uassoc()
- array_walk_recursive()
- boolval()
- file_get_contents()
- file_put_contents()
- gethostname()
- header_remove()
- headers_list()
- hex2bin()
- image_type_to_extension()
- imageflip()
- lcfirst()
- mysqli_fetch_all()
- scandir()
- str_getcsv()
- str_split()
- stream_resolve_include_path()
- strpbrk()
- substr_compare()
