PHPModernizr
============

A server-side modernizr library that gives the abiility to use last built-in PHP functions on old PHP versions.
Can be helpful to work on old PHP 4 servers.

- array_combine()
- array_diff_uassoc()
- array_udiff()
- array_udiff_assoc()
- array_udiff_uassoc()
- array_fill_keys()
- image_type_to_extension()
- imageflip()
- file_get_contents()
- file_put_contents()
- str_split()
- strpbrk()
- substr_compare()
- lcfirst()
- hex2bin()
- scandir()
- boolval()
