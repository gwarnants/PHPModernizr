PHPModernizr
============

A server-side modernizr library that gives the ability to use last built-in PHP functions on old PHP versions.
Can be helpful to work on old PHP 4 servers.

_/** @since PHP 4.3 */_
- file_get_contents()
_/** @since PHP 5 */_
- array_combine()
- array_diff_uassoc()
- array_udiff()
- array_udiff_assoc()
- array_udiff_uassoc()
- file_put_contents()
- scandir()
- str_split()
- strpbrk()
- substr_compare()
_/** @since PHP 5.2 */_
- array_fill_keys()
- image_type_to_extension()
_/** @since PHP 5.3 */_
- lcfirst()
_/** @since PHP 5.4 */_
- hex2bin()
_/** @since PHP 5.5 */_
- imageflip()
- boolval()
